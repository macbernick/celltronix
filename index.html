Celltronix

Download this project as a .zip file Download this project as a tar.gz file

Celltronix is a cellular automata that can simulate some electronic stuff by drawing PNP/NPN gate in a grid. This is inspired from the flash game Kohctpyktop.

This is very early release, only the editor is working (and need a huge amount of improvement !), and no simulation can run yet.
