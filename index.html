    • Celltronix

    Project maintained by macbernick Hosted on GitHub Pages — Theme by mattgraham

    Celltronix can simulate some electronic stuff by drawing PNP/NPN gate in a grid. This is inspired from the flash game Kohctpyktop.

    This is very early release, only the editor is working (and need a huge amount of improvement !), and no simulation can run yet.

    Written with Mono using Gtk# and Cairo.
